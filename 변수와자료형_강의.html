📦 변수란?

값을 저장하고 꺼내 쓰는 이름이 붙은 서랍장입니다. =는 수학의 등호가 아니라 "저장"을 의미합니다.

name
'정수연'
age
17
height
162.7
glasses
False
1name = '정수연'
2age = 17
3height = 162.7
4glasses = False
= 는 "저장"
오른쪽 값을 왼쪽 이름에 저장합니다. 수학의 등호와 다릅니다!
새 값을 대입하면?
기존 값이 사라지고 새 값과 자료형으로 완전히 바뀝니다.

🗂 자료형의 종류

구분자료형예시특징
단일형int-100, 0, 42소수점 없는 정수
단일형float3.14, -0.5소수점 포함 실수
단일형str'hello', '정보'따옴표로 감싼 문자열
단일형boolTrue, False참/거짓 두 가지만
컨테이너list[1, 2, 3]대괄호, 순서 있음, 수정 가능
컨테이너tuple(1, 2, 3)괄호, 생성 후 수정 불가
컨테이너dict{'a': 1}키:값 쌍
