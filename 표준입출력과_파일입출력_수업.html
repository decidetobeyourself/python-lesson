02 / 11

print() 특징

괄호 안의 내용을 화면에 출력

','(콤마)로 여러 값 동시 출력 가능

출력 후 자동으로 줄 바꿈

변수·수식도 함께 출력 가능

🧐 a * b를 print에 넣으면 어떤 값이 출력될까요?
1fruit = '청포도'
2print('내가 좋아하는 과일은', fruit, '입니다.')
3a = 7
4b = 5
5print(a,'*',b,'의 계산 결과는',a*b,'입니다.')
▶ 실행 결과
내가 좋아하는 과일은 청포도 입니다.
7 * 5 의 계산 결과는 35 입니다.
💡 콤마(,)로 구분하면 자동으로 공백 한 칸이 삽입됩니다.

표준 입력 — input() 함수
03 / 11

input() 특징

괄호 안 문자열을 프롬프트로 표시

입력받은 값을 항상 문자열(str)로 저장

반드시 변수에 대입해서 사용

⚠️ input()은 항상 str로 반환합니다.
숫자 계산 시 int() 또는 float()으로 형 변환!
1age = input('몇 살이세요?')
2print('당신은', age, '살이군요.')
▶ 실행 결과
몇 살이세요? 19
당신은 19 살이군요.

📌 형 변환 예시

# 정수로 변환
int(input('나이: '))
# 실수로 변환
float(input('키: '))

파일입출력이란?
04 / 11

파일입출력은 왜 필요할까?

💾 필요한 상황

파일로 저장된 대량 데이터를 읽어 올 때

결괏값을 파일로 저장해서 나중에 다시 쓸 때

📂 파일 처리 3단계

open() → 읽기/쓰기 → close()

파일을 열고 → 작업하고 → 반드시 닫는다

# 파일 열기
파일변수 = open(파일명, 모드)
# 작업 완료 후 반드시 닫기
파일변수.close()

파일입출력 모드:

'r' — 읽기 'w' — 쓰기 'a' — 추가
개념 ④
파일 처리 함수 정리
05 / 11
모드함수기능 설명
읽기
'r'		 read() 파일의 전체 내용을 하나의 문자열로 반환
readline() 한 줄씩 읽고 포인터를 다음 줄로 이동
readlines() 나머지 모든 줄을 읽어 리스트로 반환
'w'
'a'		 write() 파일 포인터 현재 위치에 내용 작성
close() 모든 모드에서 작업 후 반드시 호출!
⚠️ 'w' 모드로 열면 기존 내용이 모두 삭제됩니다. 이어 쓰려면 반드시 'a' 모드를 사용하세요!

① 파일 쓰기 (w 모드)

1f = open('연인의_곁.txt', 'w')
2f.write('태양이 바다에 미광을 비추면\n')
3f.write('나는 그대를 생각하노라.\n')
4f.write('희미한 달빛이 샘물 위에 떠 있으면\n')
5f.write('나는 그대를 생각하노라.\n')
6f.close()
💡 \n은 줄 바꿈 문자. write()는 자동 줄 바꿈이 없어요!

② 파일 읽기 (r 모드)

1f = open('연인의_곁.txt', 'r')
2a = f.read()
3print(a)
4f.close()
▶ 실행 결과
태양이 바다에 미광을 비추면
나는 그대를 생각하노라.
희미한 달빛이 샘물 위에 떠 있으면
나는 그대를 생각하노라.

🧩 input()으로 받은 값의 자료형은?
int(a) * int(a)에서 왜 int()가 필요할까요?

컴퓨팅 사고 포인트

입력: 변의 길이 (문자열로 받음)

처리: int() 형변환 후 제곱 계산

출력: print()로 결과 출력

1print('정사각형의 넓이를 계산하겠습니다.')
2a = input('한 변의 길이를 입력하세요: ')
3print('변의 길이', a, '의 넓이는',
int(a)*int(a), '입니다.')
▶ 실행 결과
정사각형의 넓이를 계산하겠습니다.
한 변의 길이를 입력하세요: 5
변의 길이 5 의 넓이는 25 입니다.
TypeError: can't multiply sequence by non-int of type 'str'

문자열은 * 곱셈 불가 → 반드시 int() 변환!
