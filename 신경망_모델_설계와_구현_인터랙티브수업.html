1

데이터 준비 & 특징·타깃 선정

데이터프레임 생성 → 특징(X: sweet, crunchy), 타깃(y: class) 분리

입력 특징 수 = 입력층 노드 수. 이 예제는 2개 특징 → 입력층 2개 노드.
타깃 클래스 수 = 출력층 노드 수. 과일/단백질/채소 3개 → 출력층 3개 노드.

2

신경망 모델 생성 (설계 → 구현)

층 수·노드 수·활성화 함수 결정 → Sequential + Dense로 구현

은닉층 노드 수는 데이터 복잡도에 따라 조정. 이 예제: 32 → 16.
은닉층 활성화 함수: ReLU(음수 → 0, 양수 → 그대로)
출력층 활성화 함수: Softmax(확률값 합 = 1)

3

모델 환경 설정 (compile)

손실함수 + 최적화 함수 + 평가 지표 설정

손실함수: 예측값과 실젯값의 차이 측정
→ 다중 분류: sparse_categorical_crossentropy
최적화 함수: 손실 최소화 방향으로 가중치 조정
→ 기본 선택: adam

4

모델 학습 → 평가 → 예측

fit() 으로 학습 → evaluate()로 성능 확인 → predict()로 예측

순전파: 입력 → 예측값 출력
역전파: 예측값 - 실젯값 오차 → 가중치 업데이트
이 과정을 epochs 수만큼 반복. 반복할수록 loss↓ accuracy↑

케라스(Keras) 코드 실습

식품 분류 신경망 — 단계별 코드 따라 치기 (Google Colab)

# ① 데이터 준비 — 식품 데이터프레임 생성
import pandas as pd
import numpy as np

data = {
    'sweet':   [9, 3, 8, 2, 7, 1, 4, 6],
    'crunchy': [1, 2, 3, 9, 2, 7, 8, 4],
    'class':   [0, 1, 0, 2, 0, 2, 2, 1]
    # 0=과일, 1=단백질, 2=채소
}
df = pd.DataFrame(data)

# 특징(X)과 타깃(y) 분리
X = df[['sweet', 'crunchy']].values
y = df['class'].values

print(X.shape)  # (8, 2) — 8행 2열
print(y)        # [0 1 0 2 0 2 2 1]
💡 X.shape = (8, 2) → 입력층 노드 수 = 2개 (sweet, crunchy)
# ② 신경망 모델 설계 (Sequential + Dense)
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

model = Sequential()  # 층을 순차적으로 쌓는 모델

# 첫 번째 은닉층 — 노드 32개, 활성화 함수 ReLU
# input_dim: 입력층 노드 수 (특징 수와 동일)
model.add(Dense(32, activation='relu', input_dim=2))

# 두 번째 은닉층 — 노드 16개, 활성화 함수 ReLU
model.add(Dense(16, activation='relu'))

# 출력층 — 노드 3개 (클래스 수), 활성화 함수 Softmax
model.add(Dense(3,  activation='softmax'))

model.summary()  # 모델 구조 확인
💡 출력층 노드 수 = 클래스 수 (과일0, 단백질1, 채소2 → 3개)
# ③ 모델 환경 설정 (compile)
model.compile(
    loss    = 'sparse_categorical_crossentropy',
    # 정수 라벨(0,1,2)을 그대로 쓸 때 사용하는 손실함수

    optimizer = 'adam',
    # 가중치를 빠르게 업데이트하는 최적화 알고리즘

    metrics = ['accuracy']
    # 학습 과정에서 정확도를 측정·출력
)
💡 compile = 설계도에 엔진 장착. 아직 학습이 시작된 게 아니에요!
# ④ 모델 학습 (fit)
history = model.fit(
    X, y,
    epochs = 100,  # 전체 데이터를 100번 반복 학습
    verbose = 1   # 학습 진행 상황 출력
)

# 학습 과정: loss가 줄고 accuracy가 올라가는지 확인!
# Epoch 1/100: loss=1.1 accuracy=0.375
# Epoch 50/100: loss=0.5 accuracy=0.875

# 모델 평가 (evaluate)
loss, acc = model.evaluate(X, y)
print(f"손실값: {loss:.4f}, 정확도: {acc:.4f}")
💡 epochs가 많을수록 학습 반복 횟수 ↑ → loss ↓, accuracy ↑ (단, 과적합 주의!)
# ⑤ 새 데이터 예측 (predict)
new_food = np.array([[8, 2]])  # sweet=8, crunchy=2

proba = model.predict(new_food)
print("클래스별 확률:", proba)
# 예: [[0.82, 0.10, 0.08]] → 과일(0)일 확률 82%

predicted_class = np.argmax(proba)  # 가장 높은 확률의 인덱스
labels = ['과일', '단백질', '채소']
print(f"예측 결과: {labels[predicted_class]}")
💡 predict()는 각 클래스의 확률을 출력. argmax()로 가장 높은 확률 클래스를 선택!

📋 Keras 핵심 함수 요약

함수역할주요 파라미터
Sequential()층을 순차 연결하는 모델 생성—
Dense(n, activation, input_dim)완전 연결 계층 추가n=노드수, activation, input_dim
model.compile()학습 환경 설정loss, optimizer, metrics
model.fit()모델 학습X, y, epochs
model.evaluate()모델 성능 평가X, y → loss, accuracy
model.predict()새 데이터 예측new_X → 클래스별 확률

MNIST 손글씨 숫자 분류

실제 데이터셋으로 신경망 모델 구현 + 내가 쓴 숫자 인식 실습

🎙 교사 안내

"이제 실제 데이터셋인 MNIST로 실습합니다. MNIST는 0~9 손글씨 숫자 이미지 70,000장입니다. 한 이미지는 28×28 픽셀, 총 784개 값을 가집니다."

📊 MNIST 데이터셋 특성

  • 이미지 크기: 28 × 28 픽셀 = 784개 입력값
  • 픽셀값: 0(검은색) ~ 255(흰색)
  • 클래스: 0~9 숫자 → 출력층 10개 노드
  • 전처리: 픽셀값 ÷ 255 → 0~1 정규화
  • flatten: 28×28 → 784 1차원 배열

🏗️ MNIST 모델 구조

  • 입력층: 784개 노드 (28×28 flatten)
  • 은닉층 1: 128개 노드, ReLU
  • 은닉층 2: 64개 노드, ReLU
  • 출력층: 10개 노드, Softmax
① 데이터 로드
② 전처리
③ 모델 구성
④ 학습·저장
# ① MNIST 데이터 로드
from tensorflow.keras.datasets import mnist

(X_train, y_train), (X_test, y_test) = mnist.load_data()

print(X_train.shape)  # (60000, 28, 28)
print(X_test.shape)   # (10000, 28, 28)
print(y_train[:5])    # [5 0 4 1 9] — 실제 정답 레이블
# ② 데이터 전처리
# 28×28 이미지 → 784 1차원 배열로 flatten
X_train = X_train.reshape(-1, 784)
X_test  = X_test.reshape(-1, 784)

# 픽셀값 0~255 → 0.0~1.0 정규화
X_train = X_train / 255.0
X_test  = X_test  / 255.0

print(X_train.shape)  # (60000, 784)
# ③ MNIST 신경망 모델 구성
model = Sequential()
model.add(Dense(128, activation='relu', input_dim=784))
model.add(Dense(64,  activation='relu'))
model.add(Dense(10,  activation='softmax'))
# 출력층 10개 = 숫자 0~9

model.compile(
    loss      = 'sparse_categorical_crossentropy',
    optimizer = 'adam',
    metrics   = ['accuracy']
)
# ④ 학습 및 모델 저장
history = model.fit(
    X_train, y_train,
    epochs          = 10,
    validation_data = (X_test, y_test)
)

# 모델 평가
loss, acc = model.evaluate(X_test, y_test)
print(f"테스트 정확도: {acc:.4f}")  # 약 97~98%

# 모델 저장 (내가 쓴 숫자 인식에 사용)
model.save('mnist_model.h5')
💾 저장된 mnist_model.h5 파일로 나중에 내가 쓴 숫자를 인식할 수 있어요!

정리 & 형성평가

핵심 개념 빈칸 채우기 + 선택형 퀴즈

📝 빈칸 채우기 — 핵심 개념 정리
1. 신경망은 · 은닉층 · 으로 구성된다.
2. 은닉층에서 주로 사용하는 활성화 함수는 이고, 다중 분류 출력층에는 를 사용한다.
3. 예측값과 실젯값의 차이를 측정하는 함수를 라 하고, 이를 최소화하는 함수를 라 한다.
4. 케라스에서 층을 순차적으로 쌓는 클래스는 이고, 모델 학습 함수는 model.() 이다.

② 입력 특징(feature)의 개수
③ 분류해야 할 클래스의 수
④ 학습 반복 횟수(epochs)
✅ 정답! 출력층 노드 수 = 분류 클래스 수. (과일/단백질/채소 → 3개, 숫자 0~9 → 10개)
Q2. model.compile()에서 다중 분류 시 적합한 손실함수는?
① mean_squared_error

③ binary_crossentropy
④ relu
✅ 정답! sparse_categorical_crossentropy는 정수 레이블(0,1,2…)의 다중 분류에 사용.
Q3. MNIST 데이터에서 이미지 한 장(28×28)을 신경망 입력으로 변환하면 몇 개의 입력 노드가 필요한가?
