핵심 개념 3
선형 vs 비선형 분류 문제
퍼셉트론(은닉층 없음)은 직선 하나로 두 클래스를 나누는 선형 분류만 가능합니다. 직선으로 나눌 수 없는 문제가 비선형 분류이며, 은닉층이 있어야 해결할 수 있습니다.
AND — 선형 분리 가능 퍼셉트론 O
OR — 선형 분리 가능 퍼셉트론 O
XOR — 비선형 퍼셉트론 X
원형 경계 — 비선형 퍼셉트론 X
왜 비선형 문제는 은닉층이 필요한가? 은닉층의 활성화 함수가 비선형 변환을 만들어 냅니다. 층을 쌓을수록 더 복잡한 결정 경계를 표현할 수 있습니다. XOR처럼 직선 하나로는 절대 분리할 수 없는 패턴도 은닉층 하나면 해결됩니다.

구조(architecture)의 개념
딥러닝 Deep Learning
DNN을 포함해 CNN, RNN, Transformer 등 심층 구조를 활용하는 기계학습의 한 분야.

구조(architecture)가 아닌 방법론(approach)의 개념.
포함 관계: 인공지능 ⊃ 기계학습 ⊃ 딥러닝 ⊃ DNN 활용. DNN은 심층 신경망이라는 구조, 딥러닝은 그 구조를 사용하는 학습 방법론입니다.

딥러닝 대표 구조
CNN — 합성곱 신경망
이미지 데이터에 특화. 합성곱(Convolution) 연산으로 이미지의 지역적 특징(에지, 형태)을 자동 추출. 이미지 분류, 객체 감지, 의료 영상 분석에 활용.
RNN / LSTM — 순환 신경망
순서가 있는 시퀀스 데이터(텍스트, 음성, 시계열)에 특화. 이전 단계의 출력을 현재 입력으로 피드백. LSTM은 장기 의존성 문제를 개선한 RNN.
Transformer — 어텐션 기반
자연어 처리 분야를 혁신한 구조. 어텐션(Attention) 메커니즘으로 입력 전체를 동시에 참고. GPT, BERT 등 현재 대형 언어 모델의 핵심 구조.
GAN — 생성적 적대 신경망
생성자(Generator)와 판별자(Discriminator)가 경쟁하며 학습. 생성자는 진짜 같은 데이터를 만들고, 판별자는 진짜와 가짜를 구별. 이미지 생성에 활용.

텐서플로 플레이그라운드 핵심 개념 정리
플레이그라운드를 열기 전에 화면에 나오는 용어를 미리 정리합니다.
Epoch (에포크)
전체 학습 데이터를 한 번 모두 사용해 학습한 횟수. 숫자가 올라갈수록 모델이 데이터를 더 많이 학습한 것. 플레이그라운드 좌측 상단에 표시됨.
Loss (손실)
모델의 예측값과 실제 정답의 차이. 낮을수록 좋음. 학습이 진행될수록 손실이 줄어들어야 정상. 플레이그라운드 우측 상단에 표시됨.
Learning rate (학습률)
가중치를 얼마나 크게 업데이트할지 결정하는 값. 너무 크면 발산, 너무 작으면 학습이 느림. 처음에는 0.03 정도 권장.
Hidden layers / Neurons
은닉층의 수와 각 층의 뉴런(노드) 수. + 버튼으로 층·뉴런 추가. 복잡한 문제일수록 층과 뉴런이 많이 필요.
