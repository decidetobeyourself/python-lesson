STEP 2 · 1단계
1950~1970년대 규칙 기반 인공지능 초기 · ‘인간의 지능’을 기계에 투입

🙋 인간의 역할

규칙 생성 — 지식과 경험을 바탕으로 규칙을 만들어 인공지능에 입력
‘만약 비가 오면 우산을 들고 나간다.’와 같은 규칙
시스템 설계 — 어떤 순서로 규칙을 적용하고 어떤 정보가 필요한지 구조·작동 방식을 설계
시스템 유지·관리 — 잘못된 결론이 나오면 오류를 파악해 규칙을 수정·추가
규칙 업데이트 — 새 지식·정보가 생기면 반영해 규칙을 갱신
※ 규칙 기반 AI에서는 인간의 역할이 매우 크다 — 사람의 지식·경험으로 규칙을 만들고 관리하는 것이 핵심.

🤖 인공지능의 역할

규칙 실행 — 사전에 정의된 규칙·알고리즘대로 작동, 주어진 정보로 결론 도출
추론 — 논리 체계를 이용해 새로운 정보·결론을 추론
명제: 모든 인간은 죽는다. 소크라테스는 인간이다.
추론: 소크라테스는 죽는다.
대표 응용 — 전문가 시스템
※ 규칙 기반 AI는 데이터로 ‘학습’하지 않고, 사람이 넣은 규칙을 실행·추론합니다.
STEP 2 · 2단계
1980~1990년대 데이터 기반 인공지능 도약 · 기계학습의 활용

🙋 인간의 역할

데이터 처리·준비 — 데이터를 수집·정제·라벨링하고, 필요하면 특징을 수동으로 추출
모델 평가·조정 — 성능을 평가하고, 부족하면 알고리즘의 파라미터를 조정하거나 다른 알고리즘을 시도
파라미터 — 최적의 훈련 모델을 구현하기 위해 학습에 사용하는 조절 가능한 값.
윤리적 고려 — 데이터 편향, 개인정보 보호 등 윤리적 문제를 고려

🤖 인공지능의 역할

데이터 처리 — 제공된 데이터에서 자동으로 패턴을 학습, 간단한 알고리즘으로 모델을 학습
모델 학습·활용 — 학습된 모델로 새 데이터의 예측·분류를 수행
※ 이 시기부터 AI가 ‘데이터로 학습’합니다. 다만 무엇을 볼지(특징)는 아직 사람이 정해 주는 경우가 많았습니다.
STEP 2 · 3단계
2000년대 이후 데이터 기반 인공지능 현재 · 딥러닝·생성형 인공지능의 발전

🙋 인간의 역할

데이터 제공 — AI가 학습할 적절한 데이터를 수집·준비
윤리적 고려 — 서비스·생성 콘텐츠가 사회적·문화적·윤리적으로 적절한지 평가, 규제·지침 제정
인간 중심 가치 유지 — 기술 발전 속에서 인간의 존재 가치·중요성을 지키고 강조
AI와의 공존 — 인간과 AI가 어떻게 통합하고 AI가 도움이 될 방안을 논의

🤖 인공지능의 역할

자동 특징 추출·학습 — 딥러닝으로 데이터에서 스스로 특징을 찾고 복잡한 패턴을 학습
1980~90년대엔 사람이 특징을 수동으로 골라 줬지만, 딥러닝은 특징 추출까지 자동으로 합니다.
모델 학습·활용 — 학습된 모델로 새 데이터의 예측·분류를 수행
콘텐츠 생성 — 학습한 데이터를 바탕으로 텍스트·이미지·음악 등을 생성
STEP 3 · 협력의 3가지 영역

인간과 AI는 이렇게 협력한다

창의성과 혁신

🧭

의사 결정 지원

인간의 주관적 판단과 AI의 객관적 데이터 분석이 서로를 보완한다.

⚙️

노동력 분배

반복적·단순 작업은 AI가, 전문성과 조율은 인간이 맡아 효율을 높인다.
